# ****💻ApiMathSchool****

 API build with Firebase for managing users of an Application. This application was created for the purpose of studies.
 
## ****✨ Stack****

- Firebase
- Express
- Cors
- React Hook Form

## **🛠️ Features**

- [ ] User authentication
- [ ] User Register
- [ ] Password Recovery with e-mail

## 🔧 ****Running****

BaseURL: https://us-central1-apimathschool-4971a.cloudfunctions.net/app

```js
Cadastro de usuario: 
POST: /users/create

Detalhe de Usuário:
GET: /get/id

Todos os Usuários:
GET: /api/getAll

Alteração de Informações:
PUT: /update/id

Deletar Usuário:
DELETE: /delete/id

```
